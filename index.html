Temasek Junior College Timetable Calendar Converter

Instructions:

  1. You can either:
    • Use the default TJC timetable that's already loaded, or
    • Upload your own XML timetable file
  2. Select a teacher from the dropdown list
  3. Choose the start date of the semester
  4. Choose the end date of the semester
  5. Specify whether the first week is an odd or even week
  6. Click "Generate Calendar" to download the ICS file

Note: The generated ICS file can be imported into various calendar applications such as:

  • Google Calendar
  • Apple Calendar
  • Microsoft Outlook
